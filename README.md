# yt-d (YouTube Download)
  
## EN
```
YT-D is a GUI shell for the yt-dlp utility, which allows you to download a video/audio/playlist from the YouTube platform in a couple of clicks.
  

When writing, the [therecipe/qt](https://github.com/therecipe/qt) library was used, which allows you to write graphical applications using [Qt](https://www.qt.io) on Golang
```
  
## RU
```
YT-D - это GUI оболочка для утилиты yt-dlp, которая позволяет в пару кликов скачать видео/аудио/плейлист с платформы YouTube.
  
При написании использовалась библиотека [therecipe/qt](https://github.com/therecipe/qt), которая позволяет писать графические приложения с использованием [Qt](https://www.qt.io) на Golang
```
